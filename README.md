# MementoAI_Coding

## 2번문제) 핸드폰 번호 가리기

### 문제설명
이브레인 모바일은 개인정보 보호를 위해 고지서를 보낼 때 고객들의 전화번호의 일부를
가립니다.
전화번호가 문자열 phone_number로 주어졌을 때, 전화번호의 뒷 4자리를 제외한 나머지
숫자를 전부 *으로 가린 문자열을 리턴하는 함수, solution을 완성해주세요.

### 제한조건
phone_number는 길이 4 이상, 20이하인 문자열입니다.

---
### 완성코드
<ol>
  <li>phone_number의 길이에서 뒷 4자리를 뺀 값만큼 "*"를 반복해서 문자열 생성</li>
  <li>phone_number에서 slice()메서드를 통해 뒷자리 4자리를 반환</li>
  <li>두 결과를 합쳐 answer값 반환</li>
</ol>
